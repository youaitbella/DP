Weiterentwicklung / Anfragen

Das InEK Datenportal ermöglicht Ihnen die Übermittlung bestimmter Daten an das InEK. Diese Funktionalität wird künftig weiter ausgebaut.

 

Sollten Sie Wünsche an die Funktionalität des InEK Datenportals haben, so scheuen Sie nicht, Ihre Anregungen dem InEK mitzuteilen. Solche Vorschläge können sich beispielsweise auf das InEK Datenportal im Allgemeinen beziehen, oder aber auf fachliche Fragestellungen, wie eine Anregung zum PEPP-Vorschlagsverfahren. Damit Ihre Vorschläge zügig bearbeitet werden können, haben wir unterschiedliche Mailadressen eingerichtet, mit denen Sie die zuständigen Personen im InEK zielgerichtet erreichen können.
