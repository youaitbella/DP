Daten-Fehler




Bei der Verarbeitung ist leider ein Daten-Fehler aufgetreten. Möglicherweise haben Sie versucht, einen Datensatz zu speichern, der zwischenzeitlich von jemand anderem, z.B. Ihrem Kooperationspartner geändert wurde.

Zur Vermeidung von Dateninkonsistenzen wurden Sie vorsorglich abgemeldet. Bitte melden Sie sich erneut an.


Sollten Sie Ihre Daten nicht für einen Kooperationspoartner freigegeben haben und dennoch diese Fehlermeldung erhalten, so senden Sie bitte eine Mail mit einer kurzen Beschreibung, wie es zu diesem Fehler kommt, an das InEK. Herzlichen Dank!
