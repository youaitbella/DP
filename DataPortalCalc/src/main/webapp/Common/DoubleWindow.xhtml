Mehrere Fenster


Das Datenportal hat die Nutzung mehrerer Fenster (dies kann auch ein zweiter Karteireiter im Browser sein) erkannt. Dies könnte zu einem nicht spezifizierten Verhalten führen und ist daher nicht zulässig. Vorsichtshalber wurden Sie deshalb automatisch abgemeldet (im anderen Fenster führt dies abhängig von der Klick-Reihenfolge zur Meldung "Timeout" oder direkt zum Login).

Sie können via Login die Arbeit in diesem Fenster fortsetzen. Bitte nutzen Sie im weiteren Verlauf nur ein Fenster.

Sollten Sie das ursprüngliche Fenster bereits geschlossen und trotzdem diese Meldung erhalten haben, kann dies an einer fehlenden Abmeldung liegen.

Einige moderne Browser bieten die Möglichkeit eines "privaten" Fensters, manchmal auch "Inkognito-Fenster" genannt. Damit wird es - abhängig vom Browser - möglich, das Datenportal gleichzeitig in zwei Fenstern (einmal "normal", einmal "privat" bzw. "inkognito") zu öffnen.
